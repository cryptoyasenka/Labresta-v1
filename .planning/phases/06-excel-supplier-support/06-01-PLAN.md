  <behavior>
    URL Detection:
    - is_google_sheets_url("https://docs.google.com/spreadsheets/d/1-4UJcVAUefqV1NuSAGljghvzu-ulXf-A/edit?gid=123") -> True
    - is_google_sheets_url("https://example.com/feed.xml") -> False
    - is_google_sheets_url("") -> False

    URL Conversion:
    - convert_google_sheets_url("https://docs.google.com/spreadsheets/d/ABC123/edit?gid=456#gid=456") -> "https://docs.google.com/spreadsheets/d/ABC123/export?format=xlsx&gid=456"
    - convert_google_sheets_url("https://docs.google.com/spreadsheets/d/ABC123/edit") -> "https://docs.google.com/spreadsheets/d/ABC123/export?format=xlsx" (no gid)
    - convert_google_sheets_url("https://example.com/feed.xml") -> raises ValueError

    XLSX Validation:
    - validate_xlsx_response(b'PK\x03\x04...') -> None (valid)
    - validate_xlsx_response(b'<!DOCTYPE html>...') -> raises ValueError("not shared publicly")
    - validate_xlsx_response(b'\x00\x00\x00') -> raises ValueError("not a valid .xlsx")

    Column Detection (against openpyxl worksheet):
    - Row with ["Название", "Бренд", "Модель", "Цена"] -> header_row=0, mapping={0:"name", 1:"brand", 2:"model", 3:"price"}
    - Row with ["#", "Name", "Brand", "Model", "Price", "Available"] -> header_row=0, mapping={1:"name", 2:"brand", 3:"model", 4:"price", 5:"available"}
    - Rows with no keywords -> (None, {})

    Product Parsing:
    - Valid row with brand+model+name+price -> dict with external_id="brand_lower|model_lower"
    - Row without brand -> skipped (category header)
    - Row without model -> skipped
    - Duplicate brand+model -> second occurrence skipped, error logged
    - Unparseable price "abc" -> product with available=False, price_cents=None
    - Missing availability column -> all products available=True
    - Price "1 234.56" -> price_cents=123456
    - Price "1234,56" -> price_cents=123456

    Model:
    - Supplier.column_mapping field exists as db.Text, nullable=True
  </behavior>

**Supplier model change:**
Add `column_mapping = db.Column(db.Text, nullable=True)` to the Supplier model in `app/models/supplier.py`. This stores JSON like `{"header_row": 2, "columns": {"0": "name", "1": "brand", "2": "model", "3": "price"}}`. Using db.Text (not db.JSON) for SQLite compatibility.

**Excel parser service (`app/services/excel_parser.py`):**

Create the service with these functions (reference RESEARCH.md examples):

1. **Constants:**
   - `GOOGLE_SHEETS_PATTERN` — compiled regex for `docs\.google\.com/spreadsheets/d/([a-zA-Z0-9_-]+)`
   - `COLUMN_KEYWORDS` — dict mapping field names to keyword lists in ukr/rus/eng (per CONTEXT.md)
   - `REQUIRED_FIELDS` — `{"name", "brand", "model", "price"}`

2. **`is_google_sheets_url(url: str) -> bool`** — check URL against GOOGLE_SHEETS_PATTERN

3. **`convert_google_sheets_url(url: str) -> str`** — extract spreadsheet ID and gid, construct `/export?format=xlsx&gid=XXX` URL. Raise ValueError if not Google Sheets.

4. **`validate_xlsx_response(raw_bytes: bytes) -> None`** — check PK magic bytes. If starts with `<` or contains `<!DOCTYPE`, raise ValueError("Google Sheet is not shared publicly"). Otherwise raise ValueError("not a valid .xlsx").

5. **`detect_columns(ws, max_scan_rows=10) -> tuple[int|None, dict]`** — scan first N rows for keyword matches. Return `(header_row_idx, {col_idx: field_name})`. A row is a header if it contains "name" + at least 1 more recognized field.

6. **`parse_excel_products(file_path: str, column_mapping: dict, header_row: int, supplier_id: int) -> tuple[list[dict], list[str]]`** — parse Excel file using confirmed mapping. Return `(products, errors)` where products is compatible with `save_supplier_products()`.
   - external_id = `f"{brand.lower()}|{model.lower()}"` (stable across row reordering)
   - Skip rows without brand or model silently (category headers)
   - Skip duplicate brand+model, log in errors list
   - Unparseable price: save with available=False, price_cents=None, log error
   - Default currency: "UAH" (Ukrainian supplier price lists)
   - Price parsing: handle comma as decimal separator, strip spaces
   - If availability column not mapped, default to available=True
   - Use `read_only=True` for openpyxl (memory efficient)

7. **`get_preview_data(file_path: str, max_rows: int = 10) -> dict`** — return `{"header_row": int|None, "detected_mapping": dict, "all_headers": list, "preview_rows": list}`. Opens workbook twice (read_only iterator consumed after detect_columns).

**Sanity check integration note (for Plan 02):** The >50% parse error threshold from CONTEXT.md will be checked in the sync pipeline, not in the parser itself. The parser returns `(products, errors)` and the caller decides.
