    1. Create `app/services/catalog_import.py` following the Research "Catalog CSV/XLS Import" patterns:

    2. Define column alias mapping (both Ukrainian and Russian variants, from Research section):
       ```python
       COLUMN_ALIASES = {
           # external_id
           'ідентифікатор_товару': 'external_id',
           'идентификатор_товара': 'external_id',
           # name
           'назва_позиції': 'name',
           'название_позиции': 'name',
           # article
           'код_товару': 'article',
           'код_товара': 'article',
           # price
           'ціна': 'price',
           'цена': 'price',
           # currency
           'валюта': 'currency',
           # brand / manufacturer
           'виробник': 'brand',
           'производитель': 'brand',
       }
       ```

    3. `normalize_header(header: str) -> str` — strip whitespace, lowercase.

    4. `map_headers(raw_headers: list[str]) -> dict[int, str]` — maps column index to internal field name via COLUMN_ALIASES.

    5. `parse_csv(file_stream, encoding='utf-8') -> list[dict]`:
       - Read CSV using `csv.reader()`. First row = headers.
       - Map headers using `map_headers()`. If `external_id` or `name` column not found, raise ValueError with clear message listing unrecognized headers (see Research Pitfall 3).
       - Try UTF-8 first, then cp1251 fallback if decode fails.
       - Return list of dicts with mapped field names.

    6. `parse_xlsx(file_path: str) -> list[dict]`:
       - Use openpyxl `load_workbook(file_path, read_only=True)`.
       - Try sheet name "Export Products Sheet" first, fall back to first sheet.
       - First row = headers. Map using `map_headers()`. Same validation as CSV.
       - Return list of dicts.

    7. `parse_catalog_file(file_path: str, filename: str) -> list[dict]`:
       - Dispatch to parse_csv or parse_xlsx based on file extension (.csv, .xls, .xlsx).
       - For .csv: open file, detect encoding with chardet, pass to parse_csv.
       - Raise ValueError for unsupported extensions.

    8. `save_catalog_products(products: list[dict]) -> dict`:
       - Upsert into PromProduct table: match on external_id (unique).
       - For each product: if external_id exists, update name/brand/article/price/currency. If new, insert.
       - Price: if present, parse as float * 100 -> int (cents). Handle empty/None gracefully.
       - Brand: use the 'brand' field from CSV if present. Don't attempt name parsing (deferred to Phase 2).
       - Return {"created": N, "updated": N, "total": N, "skipped": N} where skipped = rows missing external_id or name.
  </action>
