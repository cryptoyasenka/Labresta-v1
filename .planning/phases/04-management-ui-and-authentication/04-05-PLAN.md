       ```python
       class MatchRule(db.Model):
           __tablename__ = "match_rules"
           id = Column(Integer, primary_key=True)
           supplier_product_name_pattern = Column(String(500), nullable=False)  # exact name or pattern
           supplier_brand = Column(String(200), nullable=True)
           prom_product_id = Column(Integer, ForeignKey("prom_products.id"), nullable=False)
           created_by = Column(String(100), nullable=False)
           created_at = Column(DateTime, default=lambda: datetime.now(timezone.utc))
           is_active = Column(Boolean, default=True)
           note = Column(Text, nullable=True)

           prom_product = relationship("PromProduct")
       ```
       Add import to `app/models/__init__.py`.

    2. Add endpoints to `app/views/matches.py`:

       **GET /matches/search-catalog** — AJAX endpoint for catalog product search in modal:
       - Query param: `q` (search string, min 2 chars)
       - Search PromProduct by name ILIKE %q% or article ILIKE %q%
       - Return JSON: `[{id, name, brand, model, article, price}]` (limit 20 results)

       **POST /matches/manual** — Create manual match:
       - JSON body: `{supplier_product_id, prom_product_id, remember: bool}`
       - Create ProductMatch with status="manual", confirmed_by=current_user.name, score=100
       - If `remember=true`: create MatchRule with supplier product name + brand -> prom_product_id
       - Return JSON: `{"status": "ok", "match_id": N}`

       **GET /matches/rules** — Match rules management page:
       - Query all MatchRule entries, paginated
       - Render rules.html template

       **POST /matches/rules/<int:rule_id>/delete** — Delete a match rule:
       - Set rule.is_active = False (soft delete)
       - Return JSON or redirect

       **POST /matches/rules/<int:rule_id>/edit** — Edit a match rule:
       - Update note, prom_product_id, is_active
       - Return JSON or redirect

    3. Add manual match modal to `app/templates/matches/review.html`:

         - Each result row shows: name, brand, article, price (configurable columns via checkboxes)
         - "Запомнить на будущее" checkbox
