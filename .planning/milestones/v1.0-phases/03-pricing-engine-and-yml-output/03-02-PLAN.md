**1. Create `app/services/yml_generator.py`:**

The main function `regenerate_yml_feed()` must:

a) Query all confirmed matches with joined supplier product, prom product, and supplier data:
```python
from sqlalchemy import select
from sqlalchemy.orm import joinedload

stmt = (
    select(ProductMatch)
    .where(ProductMatch.status == "confirmed")
    .options(
        joinedload(ProductMatch.supplier_product).joinedload(SupplierProduct.supplier),
        joinedload(ProductMatch.prom_product),
    )
)
matches = db.session.execute(stmt).scalars().unique().all()
```

b) Build YML XML using lxml (follow `yml_test_generator.py` pattern):
- Root: `<yml_catalog date="YYYY-MM-DD HH:MM">`
- Shop: `<name>LabResta</name>`, `<company>LabResta</company>`, `<url>https://labresta.com</url>`
- Currencies: `<currency id="EUR" rate="1"/>`
- Offers: one `<offer>` per confirmed match

c) For each match, determine availability and price:
- Use `get_effective_discount(match.discount_percent, supplier.discount_percent)` for discount
- Use `is_valid_price(supplier_product.price_cents)` — if invalid, set available="false" and use price=0
- If `supplier_product.available == False` or `supplier_product.needs_review == True`, set available="false"
- Otherwise available="true"
- Price = `calculate_price_eur(supplier_product.price_cents, effective_discount)` — whole EUR integer

d) Build each offer element:
```xml
<offer id="{prom_product.external_id}" available="true|false">
    <name>{prom_product.name}</name>
    <price>{whole_eur_price}</price>
    <currencyId>EUR</currencyId>
    <vendorCode>{supplier_product.article}</vendorCode>
</offer>
```
- `offer id` = prom_product.external_id (per CONTEXT.md: so platform updates the correct product)
- `name` = prom_product.name (per CONTEXT.md: from prom.ua catalog, not supplier)
- Skip vendorCode element entirely if supplier_product.article is None/empty
- Price is always included even for out-of-stock (use last known price, or 0 if invalid)

e) Atomic file write:
- Ensure output directory exists: `os.makedirs(yml_dir, exist_ok=True)`
- Use `tempfile.mkstemp(suffix=".tmp", prefix="yml_", dir=yml_dir)` for temp file in SAME directory
- Write with lxml `tree.write()` including `xml_declaration=True, encoding="UTF-8", pretty_print=True, doctype='<!DOCTYPE yml_catalog SYSTEM "shops.dtd">'`
- `os.replace(tmp_path, output_path)` for atomic rename
- Wrap in try/except — on failure, clean up temp file with `os.unlink()` and re-raise
- Use `current_app.config["YML_OUTPUT_DIR"]` and `current_app.config["YML_FILENAME"]` for paths

f) Log: number of offers generated, number out-of-stock, output path.

g) Return a dict with stats: `{"total": N, "available": N, "unavailable": N, "path": output_path}`

**2. Create `app/views/feed.py`:**

```python
from flask import Blueprint, send_from_directory, current_app, abort

feed_bp = Blueprint("feed", __name__)

@feed_bp.route("/feed/yml")
def serve_yml():
    """Serve the generated YML feed — public, no auth required."""
    yml_dir = current_app.config["YML_OUTPUT_DIR"]
    yml_file = current_app.config["YML_FILENAME"]
    try:
        return send_from_directory(yml_dir, yml_file, mimetype="application/xml")
    except FileNotFoundError:
        abort(404)
```

No authentication on this route — per AUTH-03 and CONTEXT.md, the public YML URL must be accessible without login.
