# LabResta Sync

## What This Is

Веб-застосунок для автоматичної синхронізації цін і наявності товарів з XML-фідів постачальників на prom.ua (магазин LabResta). Замінює ручне оновлення 6,100+ товарів: система підтягує фіди постачальників, застосовує правила знижок, матчить товари і генерує YML-файл, який prom.ua підтягує автоматично кожні 4 години. Спроектовано для майбутнього переїзду на Хорошоп.

## Core Value

Ціни і наявність на prom.ua завжди актуальні — без ручної роботи щодня.

### Active

**Інтеграція з постачальниками:**
- [ ] Завантажувати та парсити YML-фіди постачальників за URL (перший: MARESTO — https://mrst.com.ua/include/price.xml)
- [ ] Підтримувати декілька постачальників з різними URL і правилами
- [ ] Оновлювати фіди за розкладом (мінімум раз на 4 год)

**Матчинг товарів:**
- [ ] Матчити товари постачальника з товарами prom.ua за брендом + моделлю (нечіткий пошук)
- [ ] Веб-UI для перегляду, підтвердження та корекції запропонованих матчів
- [ ] Зберігати підтверджені зв'язки (supplier_id ↔ prom_product_id)
- [ ] Обробляти знятих з виробництва / перейменованих товарів (сигналізувати, не ламати)

**Правила цін:**
- [ ] Налаштовувати знижку від роздрібної ціни постачальника для кожного постачальника (різний % для кожного)
- [ ] Перераховувати фінальну ціну: роздріб × (1 - знижка%)
- [ ] Валюта EUR (як є у постачальника і на prom.ua)

**Вихідний YML-фід:**
- [ ] Генерувати YML-файл сумісний з prom.ua імпортом (поля: назва, ціна, наявність, артикул)
- [ ] Файл доступний за публічним URL на власному хостингу
- [ ] Оновлювати тільки матчнуті товари (не перезаписувати весь каталог)

**Веб-інтерфейс:**
- [ ] Dashboard: статус синхронізації, кількість матчнутих/неспарених товарів
- [ ] Управління постачальниками: додати/редагувати URL, знижку, статус
- [ ] Матчинг: список пропозицій, підтвердити/відхилити/вказати вручну
- [ ] Логи: коли оновлювалось, що змінилось (ціни, наявність)

### Out of Scope

- Автододавання нових товарів з фіду постачальника на prom.ua — ручний контроль над каталогом
- Прямий API prom.ua (YML достатньо і платформонезалежний)
- Мобільний застосунок — тільки веб
- Хорошоп-специфічна інтеграція v1 — YML-формат сумісний, розширення пізніше

## Context

- **Магазин**: LabResta на prom.ua (ID: 2672990)
- **Масштаб**: 6,101 товарів, 5,633 опублікованих; ціни в EUR
- **Постачальники**: 5+, старт з MARESTO (~150 товарів, YML-каталог)
- **Матчинг**: поле "Код" порожнє у більшості товарів prom.ua — єдиний надійний ключ це бренд + модель з назви
- **Складність матчингу**: назви різняться ("стол холодильний 3 двері" vs "холодильний стол трьохдверний"), але бренд + заводська модель збігаються
- **Проблема застарілих товарів**: частина товарів на prom.ua вже не постачається або змінила маркування
- **Хостинг**: є власний хостинг для розгортання та роздачі XML
- **prom.ua імпорт**: підтримує URL-автооновлення кожні 4 год, обирає поля для оновлення (ціна, наявність)
- **Майбутнє**: можливий переїзд на Хорошоп — рішення має бути платформонезалежним

## Constraints

- **Хостинг**: деплой на власний хостинг користувача (PHP/FTP доступ типовий)
- **Вихідний формат**: YML сумісний з prom.ua (і Хорошоп)
- **Платформонезалежність**: не прив'язуватись до prom.ua API — вихід через файл
- **Продуктивність**: обробляти 6,100+ позицій без таймаутів
