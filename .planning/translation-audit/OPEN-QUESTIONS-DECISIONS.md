## chunk-020 OQ #1 — Tecnoeka EKF 664 UD: лид «16 уровней»

- **SKU:** 10 · **Артикул:** `1980857173` · **Бренд:** Tecnoeka
- **Тип:** genuine separate RU (`desc UA==RU` False, blknochg-type — genuine RU не переписывается без go-ahead).
- **Суть:** лид genuine RU «...на **16** уровней» противоречит спеке того же genuine RU «на **6** уровней / 6 листов». UA-лид счётчика уровней не имеет. Модель-код `EKF 664 UD` (6×600×400) = **6** уровней. `16` — вероятный копипаст из шаблона `MKF1664TS` (соседний SKU 11).
- **РЕШЕНИЕ Yana:** **Лид 16 → 6 уровней.** Исправить лид genuine RU «на 16 уровней» → «на 6 уровней» (привести лид к спеке того же genuine RU и к модель-коду).
- **Применение:** правит genuine RU тело → LIVE только позже (diff + go-ahead + safe mode).

## chunk-021 OQ #1 — GEMM BCB10 NC: суффикс `NC`

- **SKU:** 39 · **Артикул:** `2226833939` · **Бренд:** GEMM
- **Тип:** `desc UA==RU` True (тело обеих локалей идентично, содержит `GEMM BCB10 NC`).
- **Суть:** Назв UA «Апарат шокової заморозки GEMM BCB10» **без** `NC` ↔ genuine Назв RU «...GEMM BCB10 **NC**» с `NC`. Тело ОБЕИХ локалей содержит `GEMM BCB10 NC`. `NC` ≈ No-Condensing.
- **РЕШЕНИЕ Yana:** **Убрать `NC` из RU Назв + тела.** Унифицировать без `NC`: убрать `NC` из genuine RU Назв И из тела ОБЕИХ локалей (UA + RU). Итог: модель = `GEMM BCB10` везде.
- **Применение:** правит genuine RU Назв + тело обеих локалей → LIVE только позже (diff + go-ahead + safe mode).

- **Тип:** оба `desc UA==RU` False, `nm_ru==nazv_ru` (genuine separate RU, blknochg — genuine RU не переписывается без go-ahead).

## chunk-023 OQ #1 — Tefcold SS7300-P: суффикс `-P`

- **SKU:** 8 · **Артикул:** `2134351614` · **Бренд:** Tefcold
- **Тип:** `nm_ru==nazv_ru`; **тело ОБЕИХ локалей содержит `SS7300` БЕЗ `-P`** (ключевое отличие от chunk-022 OQ#2, где `-I` отсутствовал только в Назв, но тело там было идентично).
- **Суть:** Назв UA + Назв.мод UA «Стіл холодильний Tefcold SS7300» **без** `-P` ↔ genuine Назв RU + Назв.мод RU «...Tefcold SS7300**-P**» с `-P`.
- **РЕШЕНИЕ Yana:** **Убрать `-P` из genuine RU Назв.** Привести genuine RU к UA + телу ОБЕИХ локалей (всё без `-P`): genuine RU Назв + Назв.мод → «Tefcold SS7300» без `-P`.
- **Почему отличается от chunk-022 OQ#2 (там добавляли `-I` в UA):** здесь `-P` отсутствует **и в теле обеих локалей**, не только в UA-Назв → `-P` — артефакт в genuine RU Назв, а не официальная часть модели. У GS92/SA920 `-I` трактуется как официальная часть → добавляли в UA. Бренд один (Tefcold), но природа рассинхрона разная — решения сознательно расходятся.
- **Применение:** правит genuine RU Назв/Назв.мод (blknochg, LIVE) → LIVE только позже (diff + go-ahead + safe mode).
