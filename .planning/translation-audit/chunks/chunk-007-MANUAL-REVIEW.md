#### SKU 7 (Артикул 2593126591, ATA C2EFR05TT) — ⚠️ T5 surface conflict

Модель `C2EF**R**05TT` (суффикс R = Ribbed).

- UA текст: `<li>Поверхня гладка.</li>`
- UA spec: `<li>Тип поверхні: Ребриста</li>` ← совпадает с кодом модели
- RU текст: `<li>Поверхность гладкая.</li>`
- RU spec: `<li>Тип поверхности: Гладкая</li>` ← НЕ совпадает с кодом модели

По коду модели (R) поверхность должна быть **ребристой**. UA spec корректен. UA текст, RU текст и RU spec — неверны.

#### SKU 8 (Артикул 2593131209, ATA C2EFR05VV) — ⚠️ T6 body model code (АВТО-применено)
